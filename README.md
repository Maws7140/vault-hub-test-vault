# Vault Hub Test Vault

Test vault for developing and testing the Vault Hub plugin.

## Contents

- **Vault Hub Plugin** — Pre-installed in `.obsidian/plugins/vault-hub/`
- **CSS Snippets** — `glass-sidebar.css`, `minimal-scrollbars.css`
- **Dataview Notes** — Project Dashboard, Book Tracker (requires Dataview plugin)
- **Templates** — Daily Note, Meeting Note (requires Templater plugin)

## Testing

1. Open this folder as a vault in Obsidian
2. Enable the Vault Hub plugin in Settings > Community Plugins
3. Set your GitHub token in Vault Hub settings
4. Test the publish flow: Command Palette > "Vault Hub: Publish Resource"
5. Test browsing: Click the globe icon in the ribbon
